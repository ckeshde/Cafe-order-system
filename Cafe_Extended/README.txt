Cafe++ Extended Implementation


This project is an extended implementation of Cafe++.


New Features


* Customizable White Coffee
   * Patrons can now customize their White Coffee orders with different types of milk (e.g., regular, skim, soy).
* Expanded Coffee Menu
   * The coffee menu now includes additional options such as Latte, Cappuccino, and Flat White.
* Enhanced Order Management
   * The system can handle a wider variety of coffee orders and provides more detailed notifications to staff.
